# Telier Rebuild Package Final

Este pacote contém a versão consolidada dos documentos para conduzir o rebuild final do frontend do Telier.

## Estrutura
- `AGENTS.md` — regras permanentes do repositório
- `docs/rebuild/operational-masterplan.md` — plano operacional do rebuild
- `docs/rebuild/masterplan-interface-product-v2.md` — direção final de interface e produto
- `docs/rebuild/interface-implementation-guide-v2.md` — guia de implementação da rodada de interface
- `docs/rebuild/windsurf-operational-package-v2.md` — pacote operacional para o Windsurf
- `docs/rebuild/codex-review-package-v2.md` — pacote de revisão para o Codex

## Observação importante
Esta versão já inclui explicitamente a fase final de:
- abandono do legado
- limpeza geral do código
- remoção de restos de transição

O rebuild não deve ser considerado encerrado sem essa fase final aprovada.
